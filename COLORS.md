# SHOWING DIFFERENT COLORS.
## FORMATTING TEXT
 USED VARIABLE | COLOR CODE | OUTPUT
---------------|------------|--------------
 RESET | "\x1B[0m" | RESET NORMAL
 BOLD | "\x1B[1m" | BOLD WHITE
 ITALIC | "\x1B[3m" | ITALIC TEXT
 UNDERLINE | "\x1B[4m" | UNDERLINE TEXT
