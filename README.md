This is a library for using colors in c in your terminal or UI.

# COMPILE AND BUILD
## USAGE IN C
### In LINUX:
```
 #include "colors.h"
 #include <stdio.h>
 int main()
 {
 	printf("%sThis is a green color",GREEN); //this will print green color text
 }
```

## USAGE IN C++
### In LINUX:
```
#include "colors.h"
#include <iostream>
using namespace std;
int main()
{
   cout << GREEN << "This is a green color"; //this will print green color text
}
```
